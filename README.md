# Install


use composer install

create dir config/ and this two files :
- config.production.ini
- config.developpement.ini

copy this code and paste into files and set your infos:

; "filename"
user = "username"
password = "password"
host = "host"
dbName = "dbname"
driver = "driver"

execute model.sql and add role.csv
